Web Tools for UC Santa Cruz

Images, Buttons, Banners, and Callout Boxes

Use these tools to edit images, build callout boxes, and create buttons and banners for UC Santa Cruz websites, articles, and email messages.


Image Resizing and Cropping A tool to crop images into sizes used on UCSC websites, articles, and emails.

Button Builder A tool to generate email and form buttons.

Giving Button Builder A tool to generate giving buttons for donations.

Callout Box Builder A tool to generate callout boxes for UCSC articles.

Email Banner Builder A tool to generate email banners for UCSC emails.
